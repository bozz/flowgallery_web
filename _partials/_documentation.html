Documentation

jQuery plugin for image galleries with a cover flow effect.

Usage

The required markup for the image gallery is a simple unordered list of images: 

<ul id="gallery">
  <li><img src="..." title="image caption text" alt="image" /></li>
  <li><img src="..." title="image caption text" alt="image" /></li>
  <li><img src="..." title="image caption text" alt="image" /></li>
  <li><img src="..." title="image caption text" alt="image" /></li>
  <li><img src="..." title="image caption text" alt="image" /></li>
  <li><img src="..." title="image caption text" alt="image" /></li>
</ul>
For basic usage, select the appropriate list and initialize as follows: 

$("#gallery").flowGallery({
  easing: 'easeOutCubic',
  imagePadding: 6
});
The gallery works best if full width of browser window can be used. No direct CSS styling of gallery list required, the plugin sets everything up as needed. Be careful if you do any CSS styling of the list, this could break functionality. The only CSS styling needed, is for the loading image as specified by the 'loadingClass' attribute. Here is an example, adjust accordingly: 

#gallery li.loading {
  background: url(../images/spinner.gif) no-repeat center center;
}

Config Options

Property Type / Default Description
activeIndex integer (default: 0) the index of the image that should initially be loaded as active
animate boolean (default: true) should the image transitions be animated
backgroundColor string (default: "black") the background color of the border and caption
duration string|integer (default: 900) duration setting for animations
easing string (default: "linear") easing method to use for animation
enableKeyNavigation boolean (default: true) specifies if arrow keys (left and right) can be used for navigation
forceHeight false|integer (default: false) if specified, sets all images to this height
forceWidth false|integer (default: false) if specified, sets all images to this width
forwardOnActiveClick boolean (default: false) specifies if next image should be shown when clicking on active image
imagePadding integer (default: 4) the visible border around active images
loadingClass string (default: "loading") CSS class applied to <li> elements while image is loading, should be used to specify background loading image
thumbHeight "auto"|integer (default: "auto") the height of image thumbnails or 'auto', calculated based on aspect ratio from thumbWidth
thumbPadding integer (default: 3) the visible border around thumbnails
thumbTopOffset "auto"|integer (default: "auto") the top offset of all thumbnails or 'auto' to center based on initial active image
thumbWidth "auto"|integer (default: "auto") the width of the image thumbnails or 'auto', calculated based on aspect ratio from thumbHeight (value set to 100px if both thumbWidth and thumbHeight are set to 'auto')

API

  • next: move to next image
  • prev: move to previous image
